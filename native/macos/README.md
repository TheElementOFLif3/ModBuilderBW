Native macOS starter app for the planned split away from JavaFX.

Stack:
- `SwiftUI`
- `Swift 6`

Current scope:
- build request fields
- installer file name preview
- setup title preview
- shared JSON preview

Run on macOS:

```bash
cd "/Users/amarkovic/Desktop/Mod Builder/Bilder/Mod Billder/native/macos"
swift build
swift run
```
